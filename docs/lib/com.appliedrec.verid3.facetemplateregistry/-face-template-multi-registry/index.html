FaceTemplateMultiRegistry

class FaceTemplateMultiRegistry(registry: FaceTemplateRegistry<FaceTemplateVersion<Any>, Any>, otherRegistries: FaceTemplateRegistry<FaceTemplateVersion<Any>, Any>, ensureFaceTemplateCompatibility: Boolean = true) : SuspendingCloseable

Handles multiple face template registries.

Parameters

registry

Registry to handle

otherRegistries

Other registries to handle

ensureFaceTemplateCompatibility

Set to false to accept registries with faces that cannot be effectively compared. For example, if one registry has users user1, user2 and one of the other registries has users user1, user3 then there is no way to compare all users together because comparisons crossing registry boundaries are not possible.
